Screen elements are HTML like elements that can be used in a blogic screen.

Screen elements are grouped by type in the UI:

* Form elements
* Group elements
* View elements

## Deprecated elements

* Info Fields (use Info Fields v2)
* PinEdit (see FCK editor)
* Send To
* Table (use Data table)
