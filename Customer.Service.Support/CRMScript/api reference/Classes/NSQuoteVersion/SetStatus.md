If there was a problem with for instance calculation, this field is set to warning or error. Typically shown as an icon. QuoteStatus is an enum with statuses: 

* Ok=0
* OkWithInfo=1
* Warning=2
* Error=3

* **p_0:** Integer
* **Returns:** Void
