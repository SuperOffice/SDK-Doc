This element displays a sub-header.

## Configuration

| Setting | Description                          |
|:--------|:-------------------------------------|
| heading | Whether this is a main header        |
| div     | Whether to use a div for this header |
| title   | The title                            |
| url     | A URL (makes it into a link)         |
| target  | The target for the URL               |
| alt     | Alt for the URL                      |
| bgcolor | Background color                     |

## Example

```crmscript
heading = true
title = Support
bgcolor = blue
```
