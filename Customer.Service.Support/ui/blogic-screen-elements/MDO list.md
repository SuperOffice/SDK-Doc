This element creates a drop-down for any of the available MDO lists in SuperOffice CRM.

## Configuration

| Setting | Description                                |
|:--------|:-------------------------------------------|
| label   | UI label of the list                       |
| list    | The MDO list to get drop-down options from |

## Example

```crmscript
label = Project type
list = projecttype
```
