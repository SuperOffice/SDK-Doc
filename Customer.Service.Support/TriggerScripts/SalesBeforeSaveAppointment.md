# SalesBeforeSaveAppointment (1001)

Called before appointment is saved.

Input values:

* `IsNew` = the appointment new?
* `Location` = location
* `suggestedappointment_id` = suggested appointment id
* `project_id` = the project id
* `sale_id` = the sale id
